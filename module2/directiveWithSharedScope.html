It also has an attribute directive called shared attribute.

This is OK, as long as the element and attribute directives don't collide on these issues:

  • Multiple directives requesting isolated scope.
  • Multiple directives publishing a controller under the same
    • name.
  • Multiple directives declared with the transclusion option.
  • Multiple directives attempting to define a template or
    • templateURL.
